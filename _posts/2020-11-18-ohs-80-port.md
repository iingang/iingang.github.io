# OHS 11g

---





1. **.apachectl 권한 변경**

   **cd ${OHS_HOME}/ohs/bin**

   **`chown root .apachectl`**

   **`chmod 6750 .apachectl`**

   <br/>

   **<mark style="background-color: #fff5b1">apachectl 앞의 . 꼭 붙일 것!</mark>**  

   <br/>
   
   <br/>



# OHS 12cR1

---





1. **hasbind 권한 변경**

   **cd ${OHS_HOME}/oracle_common/bin**

   **`chown root hasbind`**

   **`chmod 4756 hasbind`**

   <br/>

2. **/etc/cap.ora 파일 생성**

   /tmp/cap.ora 파일을 /etc 아래로 복사

   - **`cp /tmp/cap.ora /etc/cap.ora`**

   권한 변경

   - **`chmod 644 /etc/cap.ora`**

     <br/>

3. **/tmp/cap.ora 파일 삭제**

   **`rm /tmp/cap.ora`**

   <br/>

   <br/>

# OHS 12cR2

---





1. **launch 권한 변경**

   **cd ${OHS_HOME}/ohs/bin**

   **`chown root launch`**

   **`chmod 4750 launch`**
